## ✨ Sobre o projeto

Este projeto foi desenvolvido para a profissional **Helenice Moreira**, com o objetivo de criar uma presença digital elegante, confiável e estratégica.

O site combina:

- **identidade visual premium**
- **copy pensada para conversão**
- **seções educativas animadas**
- **prova social**
- **FAQ para quebra de objeções**
- **integração com WhatsApp**

A ideia é que o site não apenas apresente os serviços, mas também **eduque o cliente**, **gere confiança** e **aumente a conversão**.
